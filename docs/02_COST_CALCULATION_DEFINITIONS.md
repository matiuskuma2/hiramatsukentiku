# 工種別計算方式定義

## 概要

本ドキュメントは、全35工種＋2カテゴリ（太陽光・オプション）について、
現行スプレッドシートの計算ロジックをシステムで再現するための定義書です。

各工種について以下を定義します：
1. 計算方式（calculation_type）
2. 参照する案件条件
3. 基本ロジック
4. 条件分岐
5. 要確認ポイント
6. 自動化難易度（A=高 / B=中 / C=低）

---

## 計算方式コード一覧

| コード | 説明 | 例 |
|-------|------|-----|
| `fixed_amount` | 固定額 | レッカー 58,000/式 |
| `per_tsubo` | 坪数 × 単価 | 金物 10,000/坪 |
| `per_m2` | 面積 × 単価 | 断熱材 4,182/m2 |
| `per_meter` | 長さ × 単価 | 左官基礎巾木 2,000/m |
| `range_lookup` | 範囲帯ルックアップ | 瑕疵担保保険（面積帯） |
| `lineup_fixed` | ラインナップ別固定額 | 木工事MOKU 2,100,000 |
| `rule_lookup` | ルール表で数量算出 | WB部材（面積帯→台数） |
| `manual_quote` | 都度見積／手入力 | 土工事、家具製造 |
| `product_selection` | 商品選択＋数量入力 | サッシ、内装建具 |
| `package_with_delta` | 標準セット＋差額管理 | 住宅設備、産廃 |
| `threshold_surcharge` | しきい値超過時加算 | 水道70m超 |

### 1. 外部敷地調査 (site_survey)

| 項目 | 内容 |
|------|------|
| 自動化難易度 | B |
| 計算方式 | `fixed_amount` + 条件付き採用 |
| 参照項目 | なし |
| 基本ロジック | 敷地調査 50,000/式（渥美設計へ依頼する場合） |

| 要確認 | 渥美設計への依頼有無を担当者確認 |

### 8. 基礎工事 (foundation)

| 項目 | 内容 |
|------|------|
| 自動化難易度 | A |
| 計算方式 | `per_m2` + `range_lookup` + `threshold_surcharge` |
| 参照項目 | building_area_m2 or total_floor_area_m2, foundation_perimeter_m |

**基本単価（面積帯）：**

| 面積帯 | 単価(/m2) |
|--------|----------|
| 60〜90m2未満 | 25,000 |
| 90m2以上 | 24,000 |

**別途：**

| 項目 | 単価 | 備考 |
|------|------|------|
| 深基礎 | 20,000/m | H150以内。それ以外は要相談 |

**補助計算：**
- 残土量（現状GL、設計GL、掘削深さから算出）
- 砕石量（養生面積、砕石厚から算出）
- いずれも「概算数量→現地確認で修正」が前提

**要確認：** 参照面積は建築面積か延床面積か。現行シートの式から判定要。

### 16. WB部材 (wb_parts)

| 項目 | 内容 |
|------|------|
| 自動化難易度 | B |
| 計算方式 | `rule_lookup` |
| 参照項目 | floor1_area_m2, floor2_area_m2, foundation_perimeter_m, roof_shape, fire_zone_type |

**数量算出：面積帯ルックアップ**

例：ハットヘルス
| 面積帯(m2) | 台数 |
|-----------|------|
| 〜46未満 | 1 |
| 46〜76未満 | 2 |
| 76〜106未満 | 3 |
| ... | ... |

**注意事項：**
- 各階の床面積で算出
- 1階は2階床面積分を差し引く場合あり
- 切妻の場合の追加ルールあり
- 防火/準防火地域で単価切り替え（標準:68,400 → 防火:89,550）

**専用ルールマスタが必要。**
